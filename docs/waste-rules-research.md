# 台灣與日本家戶垃圾分類研究

研究日期：2026-07-01

## 研究目的

本文件是 EcoLens Vision 分類規則的依據。範圍限於一般家庭常見垃圾，不涵蓋事業廢棄物、醫療廢棄物、農藥容器、化學品、瓦斯罐、噴霧罐、大型家電及裝修廢棄物。

這份資料不是法律意見。中央法規決定制度與責任範圍，實際收運方式、日期及部分可收品項仍可能由地方政府決定。程式無法確認地方差異時，必須顯示「查詢當地規則」，不得由 LLM 推測法規。

## 研究方法與採用標準

1. 來源優先順序為現行法規資料庫、中央主管機關、地方政府；不以部落格、新聞或回收商文章作為規則依據。
2. 每項程式規則以 `sourceIds` 連回本文參考資料。
3. 將「物品是否屬於制度回收範圍」和「居民如何投放」分開判斷。
4. 日本的實際分類由市町村執行，因此以中央制度確認範圍，再以新宿、橫濱、大阪資料驗證地方差異。
5. 只把跨來源一致且不會增加危險的前處理寫成確定指令；無法確認材質、污染程度或所在地時保留例外與不確定性。

## 主要結論

### 台灣

- 《廢棄物清理法》第 15 條建立應回收物品與容器的責任回收制度，實際品項由中央主管機關公告（TW-01）。
- 環境部目前列出的應回收類別包含鋁、鐵、玻璃、紙及塑膠容器，也包含乾電池、電子電器與資訊物品（TW-02）。
- 家戶排出前的共同處理原則是去除內容物與殘渣；容器清潔瀝乾，紙類移除非紙材。鋰二次電池要避免高溫、端子絕緣並單獨回收（TW-03）。
- 紙容器與一般廢紙應分開。複合材質、嚴重污染或低回收價值紙材可能不適合紙類回收，須依地方清運單位規定（TW-04、TW-05）。
- 「電子電器」及「資訊物品」是公告類別，但不代表照片中的每一種電子產品都適用相同管道。程式只提供指定回收方向，不宣稱固定收運方式（TW-02、TW-03）。
- 新北市的地方指引進一步使用「廢金屬容器、廢玻璃容器、廢紙容器、廢紙、廢塑膠容器」等民眾可直接投放的分類名稱；程式採用這些名稱呈現細分類，但地方收運方式仍以使用者所在地為準（TW-06）。

### 日本

- 《容器包装リサイクル法》採消費者分類排出、市町村分類收集、事業者再商品化的分工。中央制度涵蓋玻璃容器、PET 瓶、紙製及塑膠製容器包裝等（JP-01、JP-02）。
- 法律建立制度，不等於全日本居民使用同一垃圾桶或同一收運分類；居民仍需依市町村規則排出（JP-01）。
- 新宿、橫濱與大阪都將 PET 瓶與瓶蓋／標籤分開，但投放容器、收運集合方式與塑膠製品納入範圍不同，足以證明國家層級規則不能取代市町村資料（JP-05、JP-06、JP-07）。
- 小型家電有國家再資源化制度，但各市町村決定實際回收品項及方式，可能採回收箱、定點、清運站或其他方法（JP-03、JP-04）。
- 電池種類的處理方法差異很大。以大阪為例，筒型乾電池、鈕扣電池、鋰離子電池已有不同管道；程式不可只用一個「資源垃圾」指示涵蓋全部（JP-08）。

## 不確定性與程式邊界

- 國家選擇只能決定規則集合，不能精確決定日本市町村或台灣清運隊的收運日與投放地點。
- LLM 只辨識物品並回傳受控 `ruleKey`；目的地及處理文字全部由本地 JSON 規則提供。
- 辨識為噴霧罐、瓦斯罐、化學品容器、膨脹或破損電池時，不應套用一般罐瓶規則，應回傳 `unknown` 或後續建立的專用危險物規則。
- `small_electronics` 不涵蓋日本家電回收法所列大型家電，也不保證所有台灣小型電子產品都屬中央公告的同一回收項目。
- 規則應至少每半年及主管機關公告重大修法時重新檢查；每次檢查更新研究日期與來源狀態。
